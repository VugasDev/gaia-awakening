## Special Notes

### Noisium — LGPL-3.0, Modrinth Exclusive

**Noisium** ([Modrinth](https://modrinth.com/mod/KuNKN7d2)) is licensed under the **GNU Lesser General Public License v3.0 (LGPL-3.0)**. The official Noisium release is published exclusively on Modrinth.

CurseForge requires that bundled mods be available natively on CurseForge (or be on their approved non-CurseForge mods list). Because the official Noisium build is not on CurseForge, any CurseForge release of Gaia Awakening references the **unofficial NeoForge port** `noisium-unofficial-forge-port` instead. This port provides the same functionality and is CurseForge-native.

The LGPL-3.0 license permits redistribution and inclusion in modpacks, provided the license terms are preserved. The license text for Noisium can be found at the [Noisium GitHub repository](https://github.com/Steveplays28/noisium).
