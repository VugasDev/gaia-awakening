# Changelog

All notable changes to Gaia Awakening are documented here, newest first. Versions
follow the pack version in `pack.toml`. The pack is currently in **alpha**.

---

## v0.1.40-alpha

**Minecraft 1.21.1 · NeoForge 21.1.228**

The current release. Headline changes:

- **Quest system rebuilt from scratch** — 19 chapters across the five tiers,
  **142 quests total** (94 main-path + 48 optional side quests), generated from a
  reproducible, validated pipeline. Side quests teach the secondary features of the
  pack's mods, not just the critical path.
- **Explorer's Compass added** — locate any structure in the world, complementing the
  existing Nature's Compass for biomes.
- **FTB Chunks minimap disabled** — JourneyMap now handles the minimap to avoid two
  overlapping map overlays. FTB Chunks is retained for chunk claiming.
- **T4 seed fix** — corrected the Mystical Agriculture tier-4 seed namespace
  (`mysticalagriculture:` base seeds) so the higher-tier farming progression works.
- Custom item textures for the Catalyst System, drill heads, and XP-economy items.

!!! note "Earlier alpha iterations"
    Versions before v0.1.40-alpha were internal development iterations and are not
    documented individually here. This is the first release with a public changelog.
