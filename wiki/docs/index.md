# Gaia Awakening

**Gaia Awakening** is a curated Minecraft 1.21.1 (NeoForge 21.1.228) modpack built around **Create** as the
central tech progression, enriched with automation (AE2, Mekanism), magic (Ars Nouveau, Apotheosis),
exploration, and quality-of-life mods. It is designed for a small group of friends, with casual-friendly
defaults and a guided quest system that takes you from punching trees all the way to a flying fortress
powered by a fusion reactor.

## Highlights

- **Catalyst System** — custom progression gate controlling access to T3 and T4 Mystical Agriculture seeds
  via Resource Catalyst and Mythic Catalyst, each with multiple distinct acquisition paths
- **Five-tier progression** — Survival basics through Endgame, with each tier group clearly separated in the
  quest book (T1 The Beginning → T5 Endgame)
- **140+ guided quests** — FTB Quests with five tier groups, cross-chapter dependencies, and optional side
  quests marked with diamond shapes
- **Curated tech and magic** — Create (plus six companion mods), AE2, Mekanism, Ars Nouveau, Apotheosis,
  PneumaticCraft, and more, all balanced around the Create-centric progression path
- **Exploration and bosses** — Tectonic world height up to Y=512, Terralith biomes, Alex's Mobs, Alex's
  Caves, Mowzie's Mobs, L'Ender's Cataclysm, When Dungeons Arise, and all YUNG's structure mods
- **Performance-tuned** — Sodium, Iris, Distant Horizons, Lithium, FerriteCore, ModernFix and ServerCore
  included; targets GTX 1060 / RX 580 and above
